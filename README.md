# E-Waste Segregation Robot

## Author
Abhinaksha B

## Overview
This project presents a semi-autonomous robot designed to detect and segregate electronic waste from general waste using sensors and automation.

## Objective
To develop a cost-effective robotic system that can automatically identify and segregate electronic waste components such as metals, plastics, and circuit boards.

## Working Principle
Waste is placed on a conveyor system and moved using motors.  
The metal detector and inductive proximity sensors identify metallic and electronic components.  
The ESP32 processes the sensor inputs and activates servo motors to divert the waste into the appropriate bin.

## Components Used
- ESP32 Wroom
- ESP32 CAM
- Metal Detection Module
- Inductive Proximity Sensors
- L298N Motor Driver
- Side Shaft Motors
- Servo Motors
- Li-ion Battery
- Breadboard
- Jumper Wires
- Wheels
